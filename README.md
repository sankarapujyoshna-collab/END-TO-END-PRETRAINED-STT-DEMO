# T13 - End-to-End Pretrained STT Demo using Whisper

## Problem Statement

The objective of this task is to run a pretrained Speech-to-Text model and obtain transcript outputs for two different audio samples. One sample contains clear speech and the other contains speech with background noise. The transcripts generated by the model and the execution time for each sample are recorded for comparison.

## Software and Tools Used

* Python 3.13
* Visual Studio Code
* OpenAI Whisper
* PyTorch
* FFmpeg
* Windows 11 Operating System

## Installation Process

First, a virtual environment was created inside the project directory.

```bash
python -m venv venv
```

The virtual environment was activated using:

```bash
.\venv\Scripts\Activate.ps1
```

Required libraries were installed using the following commands.

```bash
pip install openai-whisper
pip install torch torchvision torchaudio
```

FFmpeg was downloaded separately and added to the system PATH variable because Whisper requires FFmpeg for processing audio files.

## Description of Audio Samples

Two audio samples were used for testing.

**clean.wav**

This audio contains a clearly spoken English sentence recorded in a quiet environment.

**noise.wav**

This audio contains speech recorded with some background disturbance to observe how noise affects speech recognition.

## Execution

The program was executed using the command:

```bash
python main.py
```

The Whisper tiny model was loaded inside the Python script.

For each audio sample, the following information was displayed:

* Transcript generated by Whisper
* Processing time
* Operating system details
* Python version

## Observation

The transcript generated for the clean audio sample was almost completely accurate.

For the noisy audio sample, a few words were recognized differently, which shows that background noise can affect the quality of speech recognition.

## Conclusion

In this task, a pretrained Whisper model was successfully executed on two audio samples. The model was able to generate transcripts and processing times for both files. The experiment demonstrated the effectiveness of Whisper for speech-to-text conversion and also showed how environmental noise influences transcription accuracy.
